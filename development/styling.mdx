## Styling Approach

Trojan Marketplace uses a **hybrid styling approach**:

1. **Styled-components** - CSS-in-JS for component-specific styles
2. **Traditional CSS files** - Global styles and complex layouts

## Styled-Components

### Installation

From `package.json:14`, styled-components is included:

```json
"styled-components": "^5.3.9"
```

### Basic Usage

Import and create styled components:

```jsx
import styled from "styled-components";

const Button = styled.button`
  background-color: #304ffe;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  
  :hover {
    background-color: #536dfe;
  }
`;
```

## Traditional CSS Files

### When to Use CSS Files

Use traditional CSS for:
- Complex layouts (Grid, Flexbox)
- Global styles
- Legacy code compatibility
- Animation keyframes

## Component Styling Best Practices

<Steps>
  <Step title="Co-locate styles">
    Define styled components near where they're used, at the bottom of component files.
    
    ```javascript
    function MyComponent() {
      return <StyledDiv>Content</StyledDiv>;
    }
    
    const StyledDiv = styled.div`
      /* styles */
    `;
    
    export default MyComponent;
    ```
  </Step>

  <Step title="Use semantic naming">
    Name styled components based on their purpose:
    
    ```javascript
    // Good
    const PrimaryButton = styled.button`...`;
    const UserAvatar = styled.img`...`;
    
    // Avoid
    const Div1 = styled.div`...`;
    const BlueButton = styled.button`...`;
    ```
  </Step>

  <Step title="Group related styles">
    Keep related styled components together:
    
    ```javascript
    // All form-related styles
    const FormContainer = styled.form`...`;
    const FormInput = styled.input`...`;
    const FormButton = styled.button`...`;
    ```
  </Step>

  <Step title="Avoid !important">
    Use specificity instead of `!important` unless absolutely necessary:
    
    ```javascript
    // Only use !important for overriding third-party styles
    border-radius: 20px !important;  // From Message.js:105
    ```
  </Step>
</Steps>
