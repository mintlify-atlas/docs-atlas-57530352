## Welcome to the Trojan Marketplace

Trojan Marketplace is a full-stack web application built with React that enables USC students to seamlessly buy, sell, and trade items within the Trojan community. Built with modern web technologies and powered by a robust Java servlet backend, the platform offers a secure and intuitive marketplace experience.

<Note>
  **Meet-up Locations**: Once items are sold, meet-ups will be at RTCC Entrance 4 and Leavey Library for safe and convenient exchanges.
</Note>

## Application Routes

The application includes the following main routes defined in `src/Main.js:28-49`:

- `/` - Home landing page with marketplace introduction
- `/login` - User authentication and login
- `/create-account` - New user registration
- `/feed` - Main marketplace feed displaying all available items
- `/profile` - User profile and account management
- `/create-sale` - Create new item listings
- `/purchase-item` - Complete item purchases
- `/message` - Direct messaging with sellers

## Getting Started

Ready to explore the Trojan Marketplace? Follow these steps:

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and run `npm install` to install all required packages including React, React Router, Axios, and Styled Components.
  </Step>
  <Step title="Configure Backend">
    Set up the Java servlet backend at `http://localhost:8080/Final_Project/` to handle authentication and data requests.
  </Step>
  <Step title="Start Development Server">
    Run `npm start` to launch the development server and open the application at `http://localhost:3000`.
  </Step>
  <Step title="Explore Features">
    Create an account, browse the marketplace feed, and start listing or purchasing items!
  </Step>
</Steps>
